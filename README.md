# Sistema de Vendas 

## Visão Geral do Projeto

Este projeto, desenvolvido em **C**, foi criado com o objetivo de praticar conceitos fundamentais da linguagem e aplicar a lógica de programação em um contexto prático. 
O sistema simula o registro e gerenciamento de vendas de uma loja, permitindo o controle de clientes e de transações realizadas.

## Conceitos Aplicados
* **Estruturas (struct)** para representar clientes e vendas.
* **Vetores e manipulação de strings** para armazenamento e busca de dados.
* **Ponteiros** para manipulação de memória dinâmica e passagem de parâmetros entre funções.
* **Laços de repetição** (for, do-while) para controle de fluxo.
* **Estruturas condicionais** (if, switch) para tomada de decisão.

## O sistema realiza as seguintes operações:

* **Cadastro de Clientes e Vendas:** Permite cadastrar o nome, idade, sexo do cliente, além das informações da venda, como quantidade de itens, valor individual e total.
* **Validação de Dados:** Garante que os valores de entrada (como nome, valor do pedido e sexo) sejam válidos, solicitando a correção em caso de erro.
* **Consulta de Informações:** O usuário pode pesquisar vendas realizadas por nome do cliente para visualizar os detalhes.
* **Relatórios Estatísticos:** Exibe um resumo das vendas e permite calcular a média de valores gastos por cliente.
